# PRODIGY_WD__02
Image Generation with Pretrained Models
In this task, an AI-based image generation system was developed to create images from text prompts using deep learning models. The project was implemented in Google Colab using Python and AI image generation libraries.
The model generated realistic and creative images based on user-provided descriptions. Different prompts related to computer science and futuristic themes were tested, and the system successfully produced visually appealing AI-generated images.
This task helped in understanding:
Artificial Intelligence in image generation
Deep learning models
Prompt-based image creation
Working with AI libraries in Python
